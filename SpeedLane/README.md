# SpeedLane (Project 23/30)

SpeedLane is an exciting, fast-paced car lane game built with HTML, CSS, and JavaScript.  
Players control a car on a 3-lane road and must avoid obstacle while collecting power-ups to increase score and survivability.  
The game features multiple difficulty levels, dynamic obstacle, power-ups, and a high score system stored in the browser.

Features:
- Three difficulty modes: Easy, Medium, Hard
- Smooth road animation with moving lane lines
- Multiple types of power-ups:
  - Shield (S): Protects from collisions
  - Slow Motion (D): Reduces obstacle speed temporarily
  - Score Multiplier (M): Doubles points for a short duration
- Score, Lives, and High Score display
- Collision detection and responsive controls (Arrow Keys)
- Game Over screen with final score and high score
- High score persistence using localStorage

Tech Stack: 
HTML | CSS | JavaScript 

How to Use:
- Clone the repository: (https://github.com/gautamsonpitale17/BuildIn30Days)
- Open index.html in your browser.
- Select your difficulty level (Easy, Medium, Hard) and click Start Game.
- Use Left and Right arrow keys to switch lanes.
- Avoid obstacles to keep your lives.
- Collect power-ups for special effects and bonus points.
- The game ends when you run out of lives.
- Your high score is saved automatically in the browse.
