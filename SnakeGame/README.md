# SnakeGame (Project 24/30)

SnakeGame is a classic snake game built using HTML, CSS, and JavaScript.  
Players control a snake to collect food while avoiding collisions with walls and itself. The game features multiple difficulty levels, responsive controls, and high score tracking using browser storage.

Features:
- Three difficulty levels: Easy, Medium, Hard
- Responsive game canvas for desktop and mobile
- Arrow key controls and optional on-screen mobile controls
- Increasing game speed as score increases
- Score and High Score tracking with localStorage persistence
- Pause, Resume, and Restart functionality
- Game Over overlay showing final score and high score

Tech Stack: 
HTML | CSS | JavaScript (Canvas API)

How to Use:
- Clone the repository: (https://github.com/gautamsonpitale17/BuildIn30Days)
- Open index.html in your browser.
- Select the difficulty level: Easy, Medium, or Hard.
- Click Start to begin the game.
- Use Arrow Keys to move the snake.
- Collect food (grey squares) to increase your score.
- Avoid colliding with the snake itself or the canvas borders.
- Pause the game with the Pause button.
- Restart anytime using the Restart button or the Game Over screen.
