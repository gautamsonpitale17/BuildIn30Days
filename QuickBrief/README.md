# QuickBrief (Project 28/30)

QuickBrief is a web-based text summarization tool that allows you to quickly condense long text into concise summaries. It uses a simple frequency-based algorithm to extract the most important sentences, helping you save time while retaining key information.  

Features:
- Summarize any text by selecting the summary length (10%, 25%, 50%).
- Copy summarized text to clipboard with one click.
- Download summary as a `.txt` file.
- Reset input and output quickly.

Tech Stack: 
HTML | CSS | JavaScript 

How to Use:
- Clone the repository: (https://github.com/gautamsonpitale17/BuildIn30Days)
- Open index.html in your browser.
- Paste or write your text in the input textarea.
- Select the summary length percentage from the dropdown.
- Click Summarize to generate a concise summary.
- Copy the summary using the Copy button or download it with the Download button.
- Use Reset to clear all fields.
