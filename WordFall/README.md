# WordFall  (Project 22/30)

WordFall is a fast-paced typing game designed to test your reflexes and typing accuracy. 
Words continuously fall from the top of the screen, and the player must type them correctly before they hit the bottom. 
The game includes multiple difficulty levels, a timer, power-ups, and smooth animations, making it both fun and challenging.

Features:
- Three difficulty modes: Easy, Medium, and Hard
- Smooth word-falling animations
- Timer with blinking warning near the end
- Pause, Resume, and Restart functionality
- Final score display at game over

Tech Stack: 
HTML | CSS | JavaScript 

How to Use:
- Clone the repository: (https://github.com/gautamsonpitale17/BuildIn30Days)
- Open index.html in your browser.
- Choose your difficulty level (Easy, Medium, or Hard).
- Start typing the words as they fall from the top of the screen.
- Type quickly before the words reach the bottom to score points.
- Some words appear in gold, these are power-ups worth extra points.
- Pause or restart the game anytime using the control buttons.
- When time runs out, your final score will be displayed on the game-over screen.
